Браузер связывается по webrtc с сервером, сервер пишет с помощью ffmpeg стримы в hls.

Можно смотреть последний добавленный стрим в браузере через webrtc (минуя сервер).

Можно смотреть все добавленные стримы в hls с небольшой задержкой (node.js получает webrtc стрим и без обработки пишет его в hls формате).

  • broadcaster - стримить веб-камеру по webrtc на node.js
  • viewer - смотреть последний добавленный стрим через webrtc
  • hls-viewer - смотреть все добавленные стримы в формате hls.
